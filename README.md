# Simple Bank System

## Description

A simple banking system developed in Java using Object-Oriented Programming (OOP) concepts.

## Features

* Create Customers and Bank Accounts
* Deposit Money
* Withdraw Money
* Transfer Money Between Accounts
* Check Account Balance
* Print Transaction Statements

## Account Types

### Savings Account

* Maintains a minimum balance (500 EGP).

### Current Account

* Supports an overdraft limit.

## OOP Concepts Used

* Abstraction
* Inheritance
* Polymorphism
* Encapsulation
* Composition

## Classes

* Bank_Account (Abstract)
* Saving
* Current_account
* Customer
* Transaction
* Main

## Example Output

The application supports deposits, withdrawals, transfers, and transaction history with date and time.
